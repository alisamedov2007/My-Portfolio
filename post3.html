Date: 2025

Category: Web Development

I built this portfolio website to practice my HTML and CSS skills and to have a place where I can share my projects, blogs, and experiences. My main goal was to create a design that is simple, clean, and easy to read. I used a dark gradient background, a clear header menu, and a flex-based layout to position different sections on the page.

While working on the site, I learned more about responsive design, spacing, font styling, and how to organize HTML structure better. I also practiced using borders, hover effects, color transitions, and different layout techniques to make the page look modern. This project helped me understand how small design decisions affect the overall appearance of a website.

In the future, I want to continue improving this website by adding a projects page, animations, and maybe some JavaScript features to make it more interactive. Building this portfolio was a great experience, and it showed me how enjoyable web development can be.
